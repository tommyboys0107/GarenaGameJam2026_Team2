# Product

GarenaGameJam2026_Team2 is a 2D game developed in Unity for the Garena Game Jam 2026. The project is in early stages, built from the Universal 2D template.

## Key Details

- Game jam project — prioritize rapid iteration and working prototypes over polish
- 2D game using Universal Render Pipeline (URP) with 2D Renderer
- Default resolution: 1920×1080
- Uses the new Input System (not legacy Input Manager)
- Player actions include: Move, Look, Attack (mapped as Vector2 and Button controls)
